# README

法律関連の各種媒体の表記一覧です（随時更新）。

[有斐閣ONLINE](https://yuhikaku.com/)における表記の基準の1つとして利用を予定しています。

## [`journals.json`](./journals.json)

| フィールド | 意味 |
| :--- | :--- |
| `id` | 乱数によるID |
| `canonical_name` | 各種表示に用いる正規化名 |
| `names` | 名称（表記揺れを配列で表現） |
| `abbrevs` | 略称（表記揺れを配列で表現） |


[`validate.py`](./validate.py) によって以下を確認しています。

- 正規化名が重複していないこと
- 正規化名が名称・略称のバリエーションに含まれていること
- IDが重複していないこと
- IDの形式が統一されていること
- 名称・略称が重複していないこと
    - `金融・商事判例` の略称の1つである `金判` が別媒体の略称になっている、といったことはありません。
    - これにより、名称もしくは略称から正規化名やIDを一意に特定することができます。

## [`journals.csv`](./journals.csv)

- [`journals.json`](./journals.json) を [`validate.py`](./validate.py) によって検証し、問題がないことを確認したうえでCSV形式に変換したファイルです。
- 略称か否かを `abbrev_flag` 列で表現しています（1=略称）
